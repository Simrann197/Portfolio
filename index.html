Weather Website

The goal of this project is to use JavaScript, HTML, CSS, Node.js, and Express.js to develop a weather website. While Node.js and Express.js power the backend functionality, HTML and CSS are used to construct the frontend. In order to retrieve and present current weather information, such as temperature, humidity, dawn and sunset timings, dependent on user input, JavaScript facilitates interaction with a weather API. With the help of this integration, customers may effortlessly access precise meteorological data via a user-friendly online interface.

Employee Attrition Model

Our machine learning project centers around predicting employee attrition using a variety of algorithms such as AdaBoost, decision trees, random forests, logistic regression, and linear regression. We've implemented these models within a Flask framework to create a user-friendly web interface. Through simple HTML and CSS elements, users can input personnel data and receive attrition risk projections promptly. This project aims to provide organizations with a practical tool to anticipate and address attrition, ultimately enhancing workforce management strategies for improved productivity.

8085 Simulator

Our C++ project is a comprehensive simulator for the 8085 microprocessor, featuring debugging capabilities and support for operations like LHLD, MOVE, and STORE. It offers a virtual environment for creating, executing, and debugging assembly language programs. With a user-friendly console panel, programmers can monitor register and simulate program execution, and identify faults efficiently. This simulator empowers developers to test programs with precision, streamlining the debugging and optimization process for 8085-based projects.
